ECOBICI Demand Prediction - CDMX

Descripción
Este proyecto analiza y predice la demanda del sistema ECOBICI en la Ciudad de México utilizando técnicas de análisis de datos y machine learning.

Objetivo
Predecir la demanda de bicicletas para optimizar la disponibilidad y mejorar la toma de decisiones en movilidad urbana.

Tecnologías utilizadas
* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

Contenido
* Análisis exploratorio de datos
* Limpieza y transformación de datos
* Modelo de predicción (Random Forest / AR)
* Visualización de resultados

Resultados
Se logró identificar patrones de uso y generar modelos predictivos con buen desempeño para estimar la demanda, el modelo ganador fue Random Forest
